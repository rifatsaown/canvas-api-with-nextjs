# Canvas App For Drawing Shapes And Lines Using Mouse Events using Next.js

## Tech Stack
- Next.js
- React
- Tailwind CSS
- Canvas API
- Roughjs(Drawing Shapes like hand drawn)

## Features
- Draw Shapes
- Draw Lines
- Draw Rectangle
- Move Shapes

## Installation
- Clone the repo
- Run `yarn install`
- Run `yarn dev`
- Open `http://localhost:3000` in your browser
